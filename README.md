# 🛒 بوت متجر ديسكورد الاحترافي | Store Discord Bot

بوت ديسكورد ذكي واحترافي مبني باستخدام **Node.js** ومكتبة **Discord.js v14** لتهيئة وإدارة متجرك الإلكتروني داخل سيرفر الديسكورد تلقائياً، مع دعم متكامل لاستقبل المنتجات عبر الويب هوك (Webhook) والتسويق بالعمولة (Affiliate). مجهز بالكامل للرفع على استضافة **Railway**.

---

## 🚀 الميزات الرئيسية (Key Features)

*   **إعداد تلقائي ذكي (`/setup-store`):**
    *   أمر مخصص للمشرفين (Administrators) لإنشاء قناة المتجر وضبط صلاحياتها فوراً.
    *   يعرض نافذة منبثقة تفاعلية (**Modal**) تطلب من المشرف تخصيص اسم القناة.
    *   يقوم بتعديل صلاحيات القناة تلقائياً (منع الأعضاء من الكتابة أو إضافة تفاعلات خارجية، مع السماح بمشاهدة القناة وقراءة الأرشيف).
*   **عرض منتجات احترافي مع أزرار تفاعلية:**
    *   تُعرض المنتجات على هيئة بطاقات غنية ومصممة بعناية (**Discord Embeds**) تشمل العنوان، الوصف، السعر، وصورة المنتج.
    *   إضافة زر شراء مباشر (**Buy Now 🛒**) ينتقل بالمستخدم مباشرة إلى صفحة المنتج.
    *   تفعيل تفاعلات تلقائية (**Reactions**) مثل 👍 و 🔥 لتسهيل تفاعل الأعضاء بشكل آمن لمنع سبام الإيموجيات الخارجية.
*   **تكامل متقدم مع مواقع الويب (Webhook API):**
    *   يحتوي البوت على خادم ويب داخلي مدمج باستخدام **Express.js** يستمع للطلبات على المسار `/api/webhook/new-product`.
    *   يتيح لموقعك الإلكتروني (سواء كان ووكومرس، شوبيفاي، أو برمجة خاصة) إرسال المنتجات الجديدة تلقائياً ليتم نشرها فوراً في الديسكورد.
*   **تتبع التسويق بالعمولة (Affiliate Support):**
    *   يقوم البوت تلقائياً بإلحاق رمز التسويق بالعمولة (`ref=your_affiliate_id`) بروابط الشراء لتتبع المبيعات والأرباح.

---

## 📋 متطلبات التشغيل (Prerequisites)

*   تثبيت بيئة **Node.js** (إصدار 16.11.0 أو أحدث).
*   حساب مطور ديسكورد وإنشاء تطبيق (Bot Application) من خلال بوابة المطورين.
*   حساب على منصة **Railway** (أو أي استضافة أخرى) للتشغيل المستمر 24/7.
